关于

Go Walker displays documentation for Go projects on Bitbucket, Github, Launchpad and Google Project Hosting.

The source code for Go Walker is available on Github.

Add a project to Go Walker

Go Walker generates documentation from Go source code. The guidelines for writing documentation for the godoc tool apply to Go Walker and display object relationships between packages in your project.

It's important to write a good summary of the package in the first sentence of the package comment. Go Walker indexes the first sentence and displays the sentence in package lists.

To add a package to Go Walker, search for the package by import path. If Go Walker does not already have the documentation for the package, then Go Walker will fetch the source from the version control system on the fly and add the documentation, as well as all dependencies packages.

Go Walker checks for package updates once per day. You can force Go Walker to refresh the documentation immediately by clicking the refresh link at the bottom of the package documentation page.

Remove a package from Go Walker

Go Walker automatically removes packages deleted from the version control system when Go Walker checks for updates to the package. You can force Go Walker to remove a deleted package immediately by clicking the refresh link at the bottom of the package documentation page, but be aware that this operation will not delete dependencies packages in case other packages use some of them.

If you do not want Go Walker to display documentation for your package, send mail to info@gowalker.org with the import path of the path of the package that you want to remove.

Feedback

Send your ideas, feature requests and questions to info@gowalker.org. Report bugs using the Github Issue Tracker. Join the discussion at the Go Walker group.
