What is this?

Go Walker generates Go projects whole source code documentation on the fly from projects on Bitbucket, Github, Google Project Hosting and Launchpad. It's NOT just API documentation, if you want to see that, GoDoc is better place for you.

Please read the About Page for information about adding projects to Go Walker and more.
